## Accessibility Checklist

- `lang="zh-Hant"` on `<html>` — required for correct CJK rendering
- All interactive elements ≥ 44×44px touch target
- Focus: `outline: 2px solid var(--gold); outline-offset: 2px` on all focusable elements
- Hero auto-advance: `prefers-reduced-motion: reduce` → never auto-advance
- Screen reader: hero slide announces `幻燈片 {N} of {M}, 《{片名}》, {狀態}` on advance
- Sidebar panels: `aria-label` on each panel container
- Image alt: backdrop = `《{片名}》劇照`; poster = `《{片名}》海報`
- Visited links: must have distinct color from unvisited (use `--gold` for visited in nav) + underline as secondary signal (WCAG 1.4.1)
